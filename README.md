# 🔍 BFS e DFS - Implementação de Algoritmos de Busca em Grafos

> Uma implementação completa e educacional dos algoritmos **Breadth-First Search (BFS)** e **Depth-First Search (DFS)** em Java - Grupo A

## 📚 Sobre o Projeto

Este projeto apresenta uma implementação detalhada e bem estruturada dos dois algoritmos fundamentais de busca em grafos:

- **BFS (Busca em Largura)**: Exploração nível por nível usando fila
- **DFS (Busca em Profundidade)**: Exploração recursiva ou iterativa usando pilha

Perfeito para estudantes de Estruturas de Dados e Algoritmos! 🎓

## ✨ Características Principais

✅ **Implementação Completa** - Algoritmos BFS e DFS totalmente implementados  
✅ **Estruturas de Dados** - Pilha e Fila customizadas  
✅ **Representação de Grafos** - Classe Graph robusta  
✅ **Interface Intuitiva** - Entrada/saída facilitada  
✅ **Código Educacional** - Bem comentado e estruturado  
✅ **Testável** - Exemplos prontos para executar
